Contacts 5+ Changelog

Version 1.0.7 (Beta)

July 21, 2016

Summary: Small bug fixes and other tweaks.

  • Fixed bug where the contents of a previously viewed group would appear at first when opening a new group.
  • Redid the application icon.
  • Redid screenshots in the Play Store listing to reflect new color scheme.

Version 1.0.6 (Beta)

July 17, 2016

Summary: Fixed more bugs, including several critical ones.

  • Fixed no-permissions screen so that Add Contact FAB and Search option are hidden.
  • Fixed critical bug where certain contacts can be missing from the All Contacts page.
  • Fixed bug where opening a contact when searching the contacts within a group would cause the search to clear.
  • Fixed critical bug where when occasionally, on re-opening the app, none of the tabs would load and the main part of the app would be a white screen.

Version 1.0.5 (Beta)

July 13, 2016

Summary: Implemented additional bug fixes.

  • Added toggle for different contacts loading processes, a possible fix for bug where certain contacts don't load in the All Contacts tab.
  • Added ability to write out log data at any point from settings page.
  • Revamped color scheme to make the blue softer.

Version 1.0.4 (Beta)

July 4, 2016

Summary: Implemented critical bug fix.

  • Possible fix for app crashing at startup on some devices

Version 1.0.3 (Beta)

July 4, 2016

Summary: Worked on improving the app behavior when it crashes, and added an error report that can easily be sent to me.

  • Fixed bug where exception handled by UncaughtExceptionHandler would cause a fail loop, and the app would freeze at a white screen.
  • Added crash report functionality. Crash reports will be deposited on the internal storage, generally in the Android/data/io.craigmiller160.contacts5/files directory.

Version 1.0.2 (Beta)

July 3, 2016

Summary: A few more tiny changes in the run up to the first official release. This version is what has been submitted to the Play Store beta channel.

  • Replaced icon with a new one that has better licensing terms.
  • Added screenshots to the project so they can be saved.
  • Changed signing key to one with a longer validity length so it's accepted by the Play Store
  • Added other images required by the Play Store listing to the project.

Version 1.0.1 (Beta)

July 3, 2016

Summary: Small bug fixes based on final testing as 1.0 was prepared to be delivered to the Play Store.

  • Make sort order case insensitive
  • Fixed App crashes when permissions granted after initially being denied

Version 1.0 (Beta)

July 3, 2016

Summary: The first full release of this app.

  • Complete overhaul of the codebase in every possible way.
  • Fast, multi-threaded contacts loading process
  • Contact photos displayed, with default image when not available.
  • Added favorite contacts tab in main page.
  • Added button for adding a new contact.
  • Added search functionality for both the all contacts list and contacts in groups list.
  • Vastly expanded settings to customize how the app displays information.

Alpha 1

February 3, 2016

Summary: A very, very early release, high buggy and unstable.

  • View all contacts and all groups.
  • Open a group and view the contacts in it.
  • A few customization settings.
