Ballot

Ballot Creator Program

View Full Source on GitHub

The very first full Java application created in this portfolio. It was designed to provide a UI for configuring a digital election ballot, then generating that digital ballot based on the user input. It was created a mere two months into the process of learning the language, and while it's a bit crude, it also demonstrates a remarkable talent for rapidly learning and applying new concepts. Nearly everything in this program was a first attempt at implementing a major concept in the language, and it all comes together and succeeds in creating a very effective program.

Key Components

Rich and Diverse GUI

A great effort was made to use a wide range of Swing components and layouts, an exploration of all that Java's UI tools had to offer.

Customized List for Data Storage

An early attempt to repurpose one of Java's core interfaces for a custom implementation. This custom List class combines both the temporary storage and tools to save and load the data from the disk. It also serves as a critical bridge to pass information between the various components of the program.

Construct New GUI Based on User Input

The primary function of this program is to provide an interface for configuring an election ballot, and then generating the ballot based on that configuration. The ballot displays in a separate window, and assembles itself based on the options set by the creator part of the program.
