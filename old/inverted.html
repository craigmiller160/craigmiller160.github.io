The oldest project in this collection, it doesn't use Java or SQL but "Edify", a special scripting language used by Android installer scripts. This program uses an enhanced tool built on Edify called the Aroma Installer, which is a powerful Android modding tool that gives end users the ability to manually select which items out of a package they wish to install. The purpose of this program was to provide a tool for the mass installation of the popular Team Blackout series of themed apps, and it was built in cooperation with the app themers themselves. The project was posted on RootzWiki.com, where it was used by many Android enthusiasts for some time.

Key Components

Aroma Installer

This script uses the Aroma Installer, an Android tool that provides a UI in recovery mode for selecting which items to install. The selections are recorded in properties files which are used when executing the installation.

Once the items to install have ben selected, the installation process needs to happen. Partitions are mounted, files are written into them, and the appropriate permissions are set.
