My Projects

I've worked on dozens of coding projects, most of them small experiments that weren't even worth uploading to my GitHub. Highlighted here is a showcase of my best work. Personal libraries and APIs and fully-composed programs that I built, mostly just for fun. Each one demonstrates skills that I've learned from my experience.

Stock Market (w/ JavaDoc)

A stock market simulation program that utilizes real-time market data from Yahoo! Finance. Concurrent, with industry standard design patterns, database access, multi-lingual, and more.

MVP Framework (w/ JavaDoc)

A fully constructed framework that programs can be built around using the Model View Presenter pattern. Near-total separation of concerns, advanced reflection, and safe concurrency are some of the key features of this project.

Money and Currency Conversion (w/ JavaDoc)

A simple library for utilizing amounts of money of different currencies. Amounts of money can have an operation performed on them, even if they have different currency types. Automatic conversions with rates from Yahoo! Finance preserve the consistency of the values.

TS-Inverted Apps

My oldest project, an installer script for themed Android apps, created in partnership with the app theme developers. It is coded in "Edify", the Android installer scripting language.

Ballot Program

My first ever completely designed and built Java program. A bit rough around the edges, but it still manages to have a sophisticated UI, save its state via IO, and respond dynamically to user input.

Key Components:

  • Fully assembled GUI in Swing
  • IO processes to save/load the program's state to a file.
  • Generates a fully assembled secondary window UI based on configuration chosen by the user.
