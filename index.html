Money and Currency Conversion

What started as my first experiment with accessing a web based service and retrieving information transformed into a pretty awesome little library. It ties BigDecimal values to currency types, which enables a number of special operations. Essentially it allows amounts of money of various currency types to be compared, combined, and otherwise operated on by a program.

Key Components:

  • Money objects must always have a BigDecimal value and a CurrentyType. Any change to the type results in a conversion operation on the value.
  • Objects of different currency types can be used together in mathematical operations (add/subtract/multiply/divide), which will automatically perform the necessary conversions to produce the correct result.
  • Amounts of money with different currency types can be compared to one another, using the Comparable interface and propper conversions.
  • Currency conversion rates are downloaded when needed from Yahoo! Finance.
  • Currently supports 24 different currency types, but could easily support all currency types, so long as the enum constant values conform to the ISO4217 standard.

TS-Inverted Apps

My oldest project, doesn't use Java or SQL but "Edify", a special scripting language used for Android installer scripts. This script is designed to install a series of themed versions of popular Android apps, and was developed in cooperation with the theme developers. It utilizes the Android Aroma Installer (from developer Amarulz) to provide the user with a simple interface to select which apps to install, defining a series of properties files along the way. It then executes the installation process based on the items the properties files indicate were selected. Unfortuantely, I had to abandon this project when my work at the time became too busy for me to continue it.

Key Components:

  • Option to do a full GApps installation along with installing the specified apps.
  • Full slate of TeamBlackout themed apps.
  • Aroma Installer gives the user the ability to pick and choose which apps to install.
