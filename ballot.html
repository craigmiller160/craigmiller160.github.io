Ballot

Ballot Creator Program

View Full Source on GitHub

My very first full Java application. A UI, action logic, IO, and everything in between. I built this after just two months of studying the language, and it utilizes a wide range of Java features, which is a testament to my ability to rapidly learn new concepts. The program's purpose is to generate a digital ballot, based on user input through a ballot configuration UI. It's a bit on the crude side, and not well documented either, but it's a good demonstration of my evolution as a programmer.

Key Components

This was my first real GUI program, so I put effort into using a wide range of Swing components and layouts.

Customized List for Data Storage

A crude but effective class that modifies the traditional List from the Java SDK. It provides special add, set, and get methods for the various ballot components. It also integrates IO methods for saving the ballot's state.

Examples in Code:

Class/Interface Description Links
BallotDataList A crude, modified List class with special methods that are tailored for the specific elements of the ballot. Also includes IO methods for saving state to file. Source

Construct New GUI Based on User Input

The primary function of this program is to provide an interface for configuring an election ballot, and then generating the ballot based on that configuration. The ballot displays in a separate window, and assembles itself based on the options set by the creator part of the program.
