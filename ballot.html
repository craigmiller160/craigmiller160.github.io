Ballot

Ballot Creator Program

View Full Source on GitHub

My very first full Java application, designed provide a UI for configuring a digital election ballot, and then generating that digital ballot based on the user input. There are so many "firsts" in this project it would take forever to list them all. First fully composed GUI, first time using listeners, first time storing program state via IO, the list goes on. I built this after just two months of studying the language, and while it can be a bit rough at times, I can look at this code and literally see how I was evolving as a developer.

Rich and Diverse GUI

This was my first real GUI program, so I put effort into using a wide range of Swing components and layouts.

Customized List for Data Storage

My first attempt to take one of Java's core interfaces and provide my own implementation. This custom List class, complete with IO loading/saving logic, is more than a little crude. However, the lessons I learned from working on this class gave me the foundation to begin expanding my exploration of Java's core library classes, and how I could specialize them through inheritence.

Construct New GUI Based on User Input

The primary function of this program is to provide an interface for configuring an election ballot, and then generating the ballot based on that configuration. The ballot displays in a separate window, and assembles itself based on the options set by the creator part of the program.
